
# react-components

### clone 
```
git clone https://github.com/gsat-technology/react-components.git
```

### install node modules
```
cd react-components
npm install
```

### install storybook cli tool
```
npm i -g @storybook/cli 
```

### run storybook
```
npm run storybook
```

### run tests
```
npm run test
```

### deploy storybook as static app
```
npm run storybook-static
```
